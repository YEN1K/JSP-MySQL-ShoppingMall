# 📊 SSG 쇼핑몰 시스템 플로우차트

본 프로젝트는 JSP와 MySQL을 연동한 쇼핑몰 웹 애플리케이션으로,
전체 서비스 흐름을 플로우차트로 설계한 후 구현하였습니다.

---

## 1. 전체 서비스 흐름

- 사용자는 메인 페이지에 접속
- 비로그인 상태에서도 상품 목록 및 게시글 조회 가능
- 회원가입 후 로그인 시 게시글 작성, 장바구니 기능 이용 가능
- 로그아웃 시 세션 종료 및 메인 페이지 이동

---

## 2. 회원 기능 흐름

1. 회원가입 페이지에서 사용자 정보 입력
2. 입력값 검증 후 DB(member_tbl)에 회원 정보 저장
3. 로그인 시 ID/PW 검증
4. 로그인 성공 시 세션 생성
5. 로그아웃 시 세션 제거

---

## 3. 게시판 기능 흐름

1. 게시글 목록 조회
2. 로그인 사용자만 게시글 작성 가능
3. 게시글 작성 시 DB(board_tbl)에 저장
4. 작성 완료 후 게시글 목록 페이지로 이동

---

## 4. 상품 및 장바구니 흐름

1. 상품 목록 조회 (item_tbl)
2. 로그인 사용자만 장바구니 담기 가능
3. 장바구니 추가 시 cart_tbl에 저장
4. 장바구니 목록 조회 가능

---

## 5. 설계 의도

- 기능별 흐름을 명확히 구분하여 JSP 파일 구조를 설계
- DB 테이블 간 관계를 고려한 JOIN 쿼리 사용
- 사용자 권한(로그인 여부)에 따른 기능 분기 처리

본 플로우차트를 기반으로 JSP 페이지와 DB 구조를 구현하였습니다.
